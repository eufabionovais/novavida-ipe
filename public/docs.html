1. Introdução

É um serviço para consumo de dados destinado à utilização por empresas que desejam ter uma integração dos seus sistemas com a nossa solução.

Este documento exibe a lista de métodos para solução via WebService. É uma interface que permite às empresas integrarem informações do sistema de forma dinâmica a softwares internos.

2. Autenticação

Para geração do token, é necessário selecionar o método GERARTOKEN e entrar com as credenciais do usuário e aguardar o retorno (forma texto “Tamanho indefinido”). Depois de gerar o Token, basta apenas colocar a string de retorno no campo Token dos métodos que contém no final o sufixo Tk.

OBS: Não há necessidade de gerar o Token toda vez que for realizar uma consulta, pois gerado uma vez ele contém validade para a data (24horas) que foi gerado e autoriza a consulta dos métodos correspondentes a sua url (conforme acima).


Exemplo gerar TOKEN:
Link: https://wsnv.novavidati.com.br/wslocalizador.asmx/GerarTokenJson

    
      {
          "credencial": {
              "usuario": "",
              "senha": "",
              "cliente": ""
          }
      }

3. Métodos NVCheck

Método para consulta de dados cadastrais, de contato e dados gerais do documento consultado.

Link para consulta: https://wsnv.novavidati.com.br/wslocalizador.asmx/NVCHECKJson


a) Parâmetros de entrada:

    
      header
      Content-Type=application/json
      Token={token gerado}

b) NVCHECK, no corpo deve ser enviado:

    
      {
          "nvcheck": {
              "Documento": "",
          }
      }
